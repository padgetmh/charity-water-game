# Water Connect

## Overview

Water Connect is an educational puzzle game inspired by charity: water's mission of bringing clean water to communities around the world.

Players rotate pipes to connect a clean water source to a village well while avoiding contaminated pipes.

---

## Features

- Rotate pipes
- Score tracking
- Multiple levels
- Contaminated pipe challenge
- Reset Game button
- Confetti celebration
- Responsive layout
- charity: water inspired colors

---

## How to Play

1. Press Start Game.
2. Click pipes to rotate them.
3. Connect the water source to the village well.
4. Avoid contaminated pipes.
5. Deliver clean water to win.
